# 技术栈

---

- 编程语言：JavaScript
- 构建工具：vue-cli
- 前端框架：Vue 2.x
- 路由工具：Vue Router 3.x
- 状态管理：Vuex 3.x
- UI 框架：Vuetify
- CSS 预编译：Stylus / Sass / Less
- HTTP 工具：Axios/Fetch
- 可视化工具：d3 + antv
- Git Hook 工具：husky + lint-staged
- 代码规范：EditorConfig + Prettier + ESLint + Airbnb JavaScript Style Guide
- 提交规范：Commitizen + Commitlint
- 单元测试：vue-test-utils + jest + vue-jest + ts-jest
- 自动部署：GitHub Actions
